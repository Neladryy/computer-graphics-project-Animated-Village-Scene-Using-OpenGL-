# Computer Graphics Project  
### Animated Village Scene Using OpenGL

## Description
This project is a computer graphics application developed using OpenGL.  
It simulates a village environment with dynamic elements such as day/night transition, rain animation, and moving objects.

## Features
- Day and Night mode
- Rain animation
- Village scene with objects
- Smooth animation

## Technologies Used
- C++
- OpenGL (GLUT)

## How to Run
1. Open the project in CodeBlocks / any C++ IDE  
2. Compile the code  
3. Run the program
